# Backend do Desafio SONDA
## Sobre
API desenvolvida para fins de avaliação da entrevista
Desenvolvida utilizando padrão REST, utilizando de Spring e Hibernate.

## Como executar
Iniciar um banco de dados local MySQL com uma table nomeada `TesteSonda`.
Indicar username e password no `application.properties` (ou usar padrão MySQL root/root root/  ).
Rode o dump `import.sql` localizado na raiz do projeto poara popular inicialmente o banco.

Importar o projeto na IDE de sua escolha  como projeto Maven e rodar classe main (Apipplication.java)
